===Procedure for METIS_gen_cal_insdark ===

This template is used for QC purposes only and measures the dark current of the detector and any thermal emission (or leaks) from inside the cold instrument. It acquires a number NEXPO of darks with specified DIT and NDIT. The darks can be taken in parallel using the different detectors (IMG LM, IMG NQ, LMS), with a different DIT/NDIT.

This dark calibration is also referred to as instrument dark and uses the dark position only in the CFO-PP1 wheel.

++++
Setup WCU|
  * Nothing to be done
++++

++++
Setup CFO and subsystems|
  * Set ''CFO-PP1'' to DARK
  * Set ''IMG-PP1'' to DARK
++++

++++
Start Exposure|
  * Start exposures with request DIT and NDIT in all subsystems
++++

++++
End Exposure|
  * Collect headers
  * Send file to ESO Archive
++++
