\section{QC Parameters}\label{sec:qc_parameters}
The following placeholders are used in the QC parameter names:
\begin{itemize}
  \item \CODE{det} is a placeholder for either \CODE{LM}, \CODE{N}, or \CODE{IFU}
  \item \CODE{cgrph} is a placeholder for one of the coronographs
% nn suffix removed for now because it caused problems with the headers

\end{itemize}
